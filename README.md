# My Static Website

A simple **static website** built with HTML and CSS, hosted on **AWS S3** and distributed via **CloudFront**.

---

## Project Overview

This project demonstrates:

- Creating a static website with HTML and CSS.
- Hosting static websites on **AWS S3**.
- Using **AWS CloudFront** for faster global content delivery.

---

## Features

- Responsive design that works on mobile and desktop.
- Clean and simple UI.
- Button link for future navigation (`Learn More`).
